## Portfolio Projects

Here is a breakdown of the projects included in this repository. Each folder contains the underlying code script along with a detailed technical documentation file.

### 1. [Google Analytics 4 (GA4) E-commerce Data Analysis](./ga4_ecommerce_bigquery/)
- **Description:** Unpacked and flattened highly nested clickstream data schemas to analyze user session behavior, build sales funnels, and track dynamic conversion trends.
- **Key Concepts:** Nested Data Flattening (`UNNEST`), Array Manipulation (`EXISTS`), Table Wildcards (`_TABLE_SUFFIX`), Window Functions (`RANK`, `ROW_NUMBER`).
- **Tech Stack:** Google BigQuery, Standard SQL.

### 2. [Movie Data: Extraction & EDA](./movie_data_analysis/)
- **Description:** Programmatically fetched multi-decade historical film records to restructure semi-structured attributes and analyze long-term industry production growth and genre cycles.
- **Key Concepts:** Programmatic Data Fetching, JSON Schema Parsing, Data Wrangling, Time-Series Feature Engineering.
- **Tech Stack:** Python, Pandas, JSON library, Seaborn.

### 3. [Titanic Passenger Profiles: Survival Insights & EDA](./titanic_passenger_analysis/)
- **Description:** Performed comprehensive exploratory data analysis on passenger demographics to identify safety patterns, calculating explicit survival benchmarks across structural socio-economic variables.
- **Key Concepts:** Missing Value Analysis, Feature Segmentation, Data Distribution Visualizations, Correlation Mapping.
- **Tech Stack:** Python, Pandas, NumPy, Seaborn, Matplotlib.
